# Archived Launchers

This directory keeps one-off recovery and historical RSAR launch scripts so old
runs remain inspectable without presenting these files as the recommended entry
points.

Use the active wrappers under `scripts/run/` for new work:

- `scripts/run/rsar_sfodrs_full.sh`
- `scripts/run/rsar_sfodrs_domain.sh`
- `scripts/run/rsar_tent.sh`
- `scripts/run/collect_results.sh`

Same-name files left in `scripts/` are compatibility wrappers that forward here
and print a deprecation notice.
